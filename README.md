# AI Datepicker 🧠📅

A lightweight, AI-powered date picker that understands natural language like:

> “Next Friday” • “3 days from now” • “1st Monday of May” • “tomorrow” • “31st August 1990”

---

## 📦 Installation

```bash
npm install ai-datepicker
```
